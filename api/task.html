Tasks#

template<typename T, context Context, typename Allocator, bool Root>
class basic_task : private lf::unique_handle<detail::promise_type<T, Context, Allocator, Root>>#

The central class of the library, a coroutine type that represents a unit of work.

Tasks support custom static/type-erased allocators, using the leading allocator convention.

Template Parameters:

  • T – The type of value that this task will return, supports references.

  • Context – The type of execution context that this task will run on.

  • Allocator – The allocator type that this task will use, use void to type-erase the allocator.

  • Root – Whether this task is the root task.
